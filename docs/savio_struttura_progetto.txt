LEGENDA CARTELLE RISERVATE
================================================================================

Le cartelle marcate [cartella riservata] sono create e pronte nella struttura
ma non contengono ancora file Kotlin. Sono placeholder per le versioni future:

  core/common/          → Result<T> sealed class, extensions Kotlin generiche
  core/util/            → DateUtils (formattazione date IT), PriceFormatter (€ format)
  feature/shoppinglist/components/  → ItemCard animata, QuantitySelector
  feature/results/components/       → MiniStoreCard per split multi-negozio (v2.0)
  res/drawable/         → ic_store_esselunga.svg, ic_store_lidl.svg, ecc.
  res/mipmap-*/         → Launcher icon Savio (tutte le density)


================================================================================
FILE DA AGGIUNGERE PRIMA DEL PRIMO BUILD
================================================================================

  1. res/mipmap-*/ic_launcher.png + ic_launcher_round.png
     → Obbligatori per compilare: Android Studio li genera automaticamente
       tramite Image Asset Studio (tasto destro su res → New → Image Asset)

  2. local.properties (NON in git)
     → sdk.dir=/Users/TUONOME/Library/Android/sdk
       (generato automaticamente da Android Studio all'apertura del progetto)

  3. google-services.json (NON in git, opzionale in MVP)
     → Necessario solo se si decommentano Firebase Crashlytics/Analytics
       in app/build.gradle.kts. Non serve per il primo build.


================================================================================
COME APRIRE IL PROGETTO
================================================================================

  1. Decomprimi savio_android_project.zip
  2. Apri Android Studio → File → Open → seleziona la cartella savio/
  3. Attendi sync Gradle (scarica ~200MB di dipendenze al primo avvio)
  4. Crea le icone launcher: res → New → Image Asset → Text "S" colore verde
  5. Run → seleziona emulatore o device fisico
  6. L'app si avvia su OnboardingScreen, poi AreaSelection con Bologna disponibile
