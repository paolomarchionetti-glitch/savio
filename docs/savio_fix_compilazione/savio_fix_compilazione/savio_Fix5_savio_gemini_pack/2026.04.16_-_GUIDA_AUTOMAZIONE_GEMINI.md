## Come funziona Gemini nell'estrazione

Lo script usa `response_mime_type="application/json"` nella config di Gemini — questo fa sì che il modello restituisca **direttamente JSON valido**, senza markdown, senza backtick, senza testo aggiuntivo. È una funzione nativa di Gemini 2.5 che rende l'output molto più affidabile rispetto al parsing di testo libero.

```python
response = self.gemini.models.generate_content(
    model="gemini-2.5-flash-preview-04-17",
    contents=prompt,
    config=types.GenerateContentConfig(
        temperature=0.1,                        # deterministico
        max_output_tokens=2048,
        response_mime_type="application/json",  # JSON garantito
    ),
)
```

## Passare a un modello diverso (opzionale)

Cambia la variabile `GEMINI_MODEL` in cima allo script:

```python
# Veloce e gratuito — consigliato per Savio
GEMINI_MODEL = "gemini-2.5-flash-preview-04-17"

# Più potente ma limiti free tier leggermente inferiori
GEMINI_MODEL = "gemini-2.5-pro-preview-03-25"

# Ultra-economico se un giorno passi a paid
GEMINI_MODEL = "gemini-2.0-flash-lite"   # $0,075/MTok input
```

## Risoluzione problemi

### "GEMINI_API_KEY non impostata"
```bash
export GEMINI_API_KEY="AIzaSy..."
# oppure in .env:
echo "GEMINI_API_KEY=AIzaSy..." >> .env && source .env
```

### "429 Too Many Requests"
Il free tier ha un limite di 10 richieste al minuto. Lo script aspetta già 1 secondo tra le chiamate. Se aggiungi molte insegne, aumenta la `time.sleep()`:
```python
time.sleep(6)  # 6 secondi = max 10 req/min
```

### "JSON non valido da Gemini"
Rarissimo con `response_mime_type="application/json"`. Se succede, lo script lo logga, salta quell'insegna e continua con le altre. Il file non viene corrotto.

### "Nessuna offerta estratta per [insegna]"
Il sito usa JavaScript per renderizzare i contenuti. Soluzione: aggiungi Playwright per il rendering JS-side:
```bash
pip install playwright && playwright install chromium
```
Poi nel config dell'insegna aggiungi `"method": "playwright"` e implementa la funzione corrispondente.
